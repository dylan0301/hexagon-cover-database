# Admissible Set

Status: Definition

Let \(\mathcal A\subset[0,1]^3\) be the set of possible local triples \((a,b,c)\).

It is symmetric:

\[
(a,b,c)\in\mathcal A\iff (b,a,c)\in\mathcal A.
\]

In the ordered half \(a\le b\), \(\mathcal A\) is the union of these cells.

## Cell 1

\[
a+b\le1,
\]

\[
a^2+ab+b^2\le1,
\]

\[
(a+b)^4-(a+b)^2+ab\le0,
\]

\[
c^4-c^2+ac-a^2\le0.
\]

## Cell 2

\[
a+b\le1,
\]

\[
a^2+ab+b^2\le1,
\]

\[
(a+b)^4-(a+b)^2+ab\ge0,
\]

\[
((a+b)^2-1)c^2+bc-b^2\le0.
\]

## Cell 3

\[
a+b\ge1,
\]

\[
a^2+ab+b^2\le1,
\]

\[
(a^2-1)c^2+(2ab^2+b)c+(b^4-b^2)\le0,
\]

\[
c\le\frac12.
\]
