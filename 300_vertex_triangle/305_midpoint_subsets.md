# Local Midpoint Subsets

Status: Reference

For a normalized \(V_0\)-triangle, the local midpoint set is

\[
\{M_5,M_0,M_1\}.
\]

Expected exact midpoint subsets:

- Vd0 \((o,n)=(1,0)\): \(\varnothing\).
- Vd0 \((o,n)=(2,0)\): \(\{M_0\}\).
- Vd1 \((o,n)=(1,1)\):
  \[
  \varnothing,\ \{M_0\},\ \{M_1\},\ \{M_5\},\ \{M_0,M_1\},\ \{M_0,M_5\}.
  \]
- Vd2 \((o,n)=(1,2)\):
  \[
  \{M_0\},\ \{M_0,M_1\},\ \{M_0,M_5\},\ \{M_0,M_1,M_5\}.
  \]
- T3-like \((o,n)=(2,1)\):
  \[
  \{M_1\},\ \{M_5\}.
  \]
