# T3-like Constraints

Status: Lemma target

Type 3 equation:

\[
\gamma^2-\alpha\gamma+\alpha^2=((\alpha+\beta)\gamma)^2.
\]

Equivalent notation:

\[
c^2-ac+a^2=((a+b)c)^2.
\]

Important restrictions:

\[
\alpha+\gamma>1,
\qquad
\beta+\gamma>1.
\]

Otherwise the configuration is axis-aligned-coverable.

Target claim:

\[
\alpha+\beta>\frac{\sqrt3}{2}.
\]

This remains a lemma target.
