# T3-like Midpoint Lemma

Status: Lemma target

Target statements:

1. If a T3-like triangle \(T_i\) contains \(M_i\), then it cannot contain \(M_{i+1}\).
2. If \(T_i\) is T3-like, then after maximality one may assume it contains one of \(M_{i-1}\) or \(M_{i+1}\).

The second statement is more speculative and belongs to the global T3-like strategy.
