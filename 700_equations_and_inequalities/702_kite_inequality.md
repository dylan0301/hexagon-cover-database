# Kite Inequality

Status: Partial / reference

A kite tuple has the form

\[
(x,y,0,z,0,y)
\]

up to \(C_6\)-rotation.

If such a kite is coverable by a triangle, then

\[
z\le \frac{\sqrt{x^2+y^2-xy}}{x}-x.
\]

This filtered some Type \(211111\) cases but did not resolve all Type \(211311\)-like examples.
