# Outer Gamma Set Constraints

Status: Reference / failed-strategy context

The outer gamma set is the historical separating-set name for the six-tuples of vertex-triangle radial coordinates produced under edge split data

$$
b_i+a_{i+1}=1.
$$

The cyclic feasible set is

$$
S=\{(a_1,b_1,\dots,a_6,b_6):b_i+a_{i+1}=1,
\ a_i^2+a_ib_i+b_i^2\le1\}.
$$

In current notation these tuples are

$$
(c(a_1,b_1),\dots,c(a_6,b_6)).
$$

Outside this failed separation strategy, use $c_i$ or $c(a_i,b_i)$ rather than `outer gamma`.
