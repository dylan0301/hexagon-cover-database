# Open/Closed/Scaled Equivalence

Status: Reference

The main open-triangle target is equivalent to the expanded closed-triangle
target:

$$
\text{For every }L>1,\ H_L\text{ cannot be covered by seven closed unit equilateral triangles.}
$$

The proof is recorded in
[`100_foundations/103_open_unit_vs_shrunken_closed_equivalence.md`](../100_foundations/103_open_unit_vs_shrunken_closed_equivalence.md).

This scaffold uses the expanded closed formulation when a branch proves a
closed-triangle obstruction on $H_L$, $S(H_L)$, or $S_{1/2}(H_L)$, and then
refers back to the equivalence theorem for the original open formulation.
