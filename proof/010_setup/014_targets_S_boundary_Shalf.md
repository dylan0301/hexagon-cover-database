# Boundary, Skeleton, And Half-Skeleton Targets

Status: Reference

The full hexagon target concerns $H$. Some branch obstructions use the boundary
$\partial H$, the skeleton $S$, or the half-skeleton target $S_{1/2}$.

The target sets are defined in
[`100_foundations/102_targets_skeleton_half_skeleton_lotus_benzene.md`](../100_foundations/102_targets_skeleton_half_skeleton_lotus_benzene.md).

The May 24 numerical counterexample refutes standalone noncoverage of the full
skeleton $S$:

- [`800_computation/811_skeleton_cover_counterexample.md`](../800_computation/811_skeleton_cover_counterexample.md).

Therefore this scaffold treats skeleton and half-skeleton claims as local or
conditional branch obstructions unless an explicit source file proves the
corresponding implication.
