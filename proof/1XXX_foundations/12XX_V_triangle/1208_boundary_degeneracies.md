# Boundary Degeneracies

Status: Definition

Important degeneracies:

- $a=0$, $b=0$, or $c=0$;
- $a+b=1$;
- $a^2+ab+b^2=1$;
- endpoint-only coverage;
- midpoint on triangle boundary;
- tangency to a hexagon edge;
- point-only CE contact.

Open-triangle statements use strict containment. Closed-triangle computations allow boundary containment but must flag degeneracy.
