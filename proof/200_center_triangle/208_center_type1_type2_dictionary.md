# Center Type 1 / Type 2 Dictionary

Status: Reference

Both Type 1 and Type 2 use the degree-indexed ray-intersection distance tuple

$$
d_0,d_{60},d_{120},d_{180},d_{240},d_{300}.
$$

Here $d_\theta$ is the scalar distance from $O$ to the point where $\partial T_C$ meets the ray from $O$ in direction $\theta$.

Type 1 and Type 2 are different vertex-cone arrangements for the same kind of six-distance data. Their algebraic varieties should be compared through these ray-intersection distances, not through special constant tuples.
