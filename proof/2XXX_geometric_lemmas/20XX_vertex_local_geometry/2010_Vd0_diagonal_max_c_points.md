# Vd0 Diagonal Max-$c$ Points

Status: Reference

The neighboring-ray maximum formula is recorded in
[../24XX_neighbor_capacity/2400_neighbor_ray_max_c_formula.md](../24XX_neighbor_capacity/2400_neighbor_ray_max_c_formula.md).

The algorithm-2 finite-point route uses the Vd0 intuition that diagonal
max-$c$ points can be selected without the Vd1/Vd2 or T3-like complications.

To use this in a proof, one must still prove:

- the relevant Vd0 branch-realization conditions;
- that the chosen diagonal point is the first needed obstruction point;
- compatibility with nonlocal cuts by other vertex regions.

Those claims are not proved here.
