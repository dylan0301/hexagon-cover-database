# Zero-Diagonal $F$-Map

Status: Definition / notation bridge

The CE0 + Vd1/Vd2 proof uses a coverage-coordinate map

$$
F(t)=\frac{t-1+\sqrt{1+6t-3t^2}}{2}.
$$

It is determined by

$$
F(t)^2+F(t)(1-t)+(1-t)^2=1.
$$

This is the zero-diagonal propagation map in the coverage-coordinate convention.

The general $g_c$ map is written in a defect-coordinate convention. Thus $F$ should not be identified with $g_0$ without applying the coordinate dictionary.
