# Minimum Enclosing Equilateral Triangle Tools

Status: Reference

For a finite point set $K$, the finite-point routes use the optimized enclosing
side length $\Lambda(K)$. The May 25 target records this convention in
[`630_may25_five_point_conjecture/632_five_point_target_and_obligations.md`](../630_may25_five_point_conjecture/632_five_point_target_and_obligations.md).

The quadrilateral support reduction used by the May 21/22 Pattern A package is
recorded as a proven local lemma in
[`500_half_skeleton_lemmas/507_minimal_enclosing_equilateral_quadrilateral_lemma.md`](../500_half_skeleton_lemmas/507_minimal_enclosing_equilateral_quadrilateral_lemma.md).

Algorithm-2 certificates must state their active support regimes explicitly
before any monotonicity or convexity claim can be used as proof.
