# Algorithm-1 Unimodality Failure

Status: Failed

The earlier algorithm-1 route tried to use the three local diagonal maximum
points for $V_0,V_1,V_2$ and then prove a unimodality statement for the
minimum enclosing triangle side length as a function of $a_1$ or $b_1$.

The sketch records that this unimodality breaks because the support pattern of
the five-point minimum enclosing equilateral triangle changes: different
points lie on different supporting sides in different regions.

This failure does not refute the finite-point strategy itself. It records why
algorithm 1 should not be used as the organizing proof route.
